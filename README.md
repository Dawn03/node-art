# node-art

## 介绍
+ 使用node + express + art-template + jquery
+ 简单的防猫眼网站
+ 动态数据来自使用easy mock


## 食用
+ 下载到本地
	+ `git clone git@github.com:Dawn03/node-art.git` 
+ 安装
	+ `npm i`
	+ 或者使用淘宝的`cnpm i`

+ 进入根目录下运行node
	+ 开第一个命令行
	+ cd进入项目根目录的server文件夹
	+ `node app`
		+ 看到`server start at localhost:3005`即node启动成功
		+ 其中只有首页和榜单页 有内容， 首页的热映和榜单是动态数据，其他均为静态
+ 启动项目
	
+ 打开`http://localhost:3005`
